# Changelog — Spell Thief

All notable changes to this module will be documented in this file.

---

## [1.4.0] — Current

### Core Class
- **Equipment:** Changed armour options to studded leather armour or chain shirt
- **Proficiencies:** Removed spears from weapon proficiencies
- **Preparing Spells:** Updated to 5e 2024 edition style — spells prepared count detailed on class table
- **Stolen Spells:** Added restriction to transcribe only wizard spells via Arcane Pickpocket
- **Arcane Pickpocket:** Changed flat DC 14 to enemy spell save DC + spell level
- **Fighting Style:** Updated description to Fighting Style feat (2024 edition)
- **Ability Score Improvement:** Updated description from 5e-tools; changed 19th-level ASI to Epic Boon
- **Arcane Heist:** Added Redirect Magic to the list of free-cost abilities during the feature

### Arcane Reaver
- **Antimagic Strikes:** Removed redundant extra attack line

### Hex Broker
- **Condition Stacking:** Clarified that advantage applies for allies (not only disadvantage for enemies)
- **Curse Collector:** Added ability to learn stolen spells from the warlock spell list

### Mirror Savant
- **Illusory Decoy:** Rewrote description to align with mirror image spell mechanics
- **Swap Fate:** Added Illusory Decoy buff to the swapped target

---

## [1.3.0]

- Focus Specialization feature added (7th level): Status Transposition and Arcane Absorption
- Mirror Savant: Swap Fate redesigned
- Hex Broker: Plague Broker range and targeting clarified

---

## [1.2.0]

- Added Improved Arcane Larceny (11th level): spell cache expanded to two spells, advantage on low-level steals
- Grand Theft Magic (15th level): automatic steal of 4th level or lower spells
- Balance pass on Larceny Dice scaling

---

## [1.1.0]

- Larcenous Disciplines introduced: Arcane Reaver, Hex Broker, Mirror Savant
- Spell Interception added (5th level): Concentration Theft and Arcane Sense
- Arcane Reaver: Spellblade Strike, Arcane Reflex, Antimagic Strikes

---

## [1.0.0] — Initial Release

- Base Spell Thief class with 20 levels
- Core features: Arcane Pickpocket, Cunning Action, Redirect Magic, Fighting Style, Spellcasting
- Larceny Dice system (d6 → d8 → d10 → d12)
- Half-caster progression (Intelligence-based)
- Spell Thief base spell list
- Master Larcenist (17th), Elusive (18th), Arcane Heist (20th)
